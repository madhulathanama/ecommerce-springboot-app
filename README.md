# E-Commerce Web Application (Ongoing)

🚧 Project Status: Ongoing Development

This project is a full-stack E-Commerce web application being developed using Java, Spring Boot, PostgreSQL, HTML, CSS, and JavaScript.

## Planned Features
- User Registration & Login
- Product Management
- Shopping Cart
- Order Processing
- REST API Integration
- Database Connectivity

## Tech Stack
- Java
- Spring Boot
- PostgreSQL
- HTML
- CSS
- JavaScript

Current development focuses on backend modules, authentication, and database integration using Spring Boot and PostgreSQL.

Code and modules are being added progressively during development.
